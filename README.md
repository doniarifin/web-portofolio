# Web-profile-portofolio using VueJs and Tailwind

This is my web-profile-portofolio

[click here](https://my-profile-dhoni.vercel.app/) to see it

## Quick start

### 1. Instal depedencies

- npm

```
npm install
```

- yarn

```
yarn install
```

---

### 2. Run the app

- npm

```
npm run dev
```

- yarn

```
yarn dev
```
